# 🏎️ Mario Kart - Página Temática

Uma página web interativa e divertida sobre Mario Kart, desenvolvida com HTML, CSS e JavaScript.

## 📋 Sobre o Projeto

Um projeto educativo que apresenta informações sobre o jogo Mario Kart com design temático, incluindo diferentes circuitos e características do jogo de forma interativa.

## ✨ Características

- 🎨 Design temático inspirado em Mario Kart
- 📱 Interface responsiva e intuitiva
- ⚡ Interatividade com JavaScript
- 🏁 Página dedicada aos circuitos
- 🎮 Navegação fluida

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização, animações e layout responsivo
- **JavaScript** - Interatividade e funcionalidades dinâmicas

## 📁 Estrutura do Projeto

```
devclub-mario-kart/
├── index.html                 # Página principal
├── circuits.html              # Página sobre os circuitos
├── style.css                  # Estilos globais
├── scripts.js                 # Lógica JavaScript
└── assets/                    # Imagens e recursos
    └── (ícones e imagens)
```

## 🚀 Como Usar

### Pré-requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- JavaScript habilitado no navegador

### Visualizar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/BrunoRivaroli/devclub-mario-kart.git
```

2. Abra o arquivo `index.html` em seu navegador

3. Ou acesse a versão online: [GitHub Pages](https://brunorivaroli.github.io/devclub-mario-kart/)

## 🎮 Como Navegar

- **Página Principal** - Introdução e informações gerais sobre Mario Kart
- **Circuitos** - Explore os diferentes circuitos do jogo
- Use os menus de navegação para se mover entre as páginas
